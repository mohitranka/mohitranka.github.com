Projects

Apart from day jobs, some of the independent/freelance projects I have worked on in the past.

TwitterSentiment

Sentiment analysis of tweets using Bayesian classifier. A generic Bayesian Model was prepared to classify tweets to positive, negative and neutral sentiments, based on various features detected in the tweet.

The overall accuracy of the trained model was 98%. TwitterSentiment uses pickled objects created from corpora available at [http://github.com/mohitranka/TwitterSentimentCorpora].

Speechify

This chrome extension adds google searchesque speech-to-text capability to all the input boxes of the web pages.

mDataRecovery

mDataRecovery, is a tiny command-line file backup (and restore ;)) utility, based on on python and sqlite. The utility is designed to support multiple locations for the versioned backup target, like AWS, Local, and Over the network using rSync.
